## 更新履歴

### (2026/04/20) Ver 1.2.3 -> 1.2.4

## 概要
v1.2.4では、SICK製PLCを介した遠隔運転許可・非常停止・エラーリセットなどの**PLC統合機能**、**協調搬送の安定性向上**、バッテリーBMS/CANブリッジ/音声連動ドライバ等の**新パッケージ追加**、および協調後の単体移動時の状態残留・地図切替失敗後の再開不能など**協調/ナビゲーション周りのバグ修正**が主な内容です。内部的にはナビゲーションスタックのモジュール化が進行中で、旧isamベースのデッドレコニング実装が廃止されました。

### 機能追加

- SICK PLC統合機能
    - PLCから遠隔自律移動許可・禁止信号を受けて走行可否をガード（PLC未接続時はデフォルト許可）
    - PLCによる非常停止を協調グループへ自動伝搬
    - ERRORリセット用アクション追加（SLS OFFでもリセット可能に）
    - Watchdog応答なしエラー、非常停止詳細情報（EstopDetailFromPLC）をROS2トピックへ配信
    - 運転準備、セレクタ自動/手動切替、SLS OFF信号の入出力に対応
- 新規パッケージ
    - `triorb_snr_mux_driver`: SNR-MUXボードとのシリアル通信ドライバ。navigate/navigation_managerの停止・一時停止遅延を音声再生と動的連動
    - `triorb_can`: SocketCANベースのCANバス⇔ROS2ブリッジ
    - `triorb_battery_info`: CAN経由でバッテリーSOC・モジュール電圧/電流を集約し `/battery/status` を配信（直列/並列構成対応）
- ナビゲーション
    - リフター中間点（middle）移動アクション追加
    - リフター一時停止、位置/エラーの定期配信追加
    - STUBモード対応（実機なしでの動作確認用）
    - ゴール到達判定を楕円範囲に変更
    - `robot_max_vel.yaml` による速度上限の筐体別適用
    - PLCから許可するタスク種別の拡充
- カメラ
    - `AutoGainTarget.srv` による自動ゲイン目標値のランタイム変更（Argus/UVC両対応）
    - 複数カメラを結合したスタックド画像トピック（1/3/4カメラ結合）
    - AWB・明るさ変更タスクの追加
    - No-Signal時のカメラノード自動再起動
    - Argusデバイス異常後の自動復活
- リモコン
    - `triorb_gamepad` のLifecycleノード化
    - Microsoft Xbox Controller用パラメータファイル追加
    - ゲームパッド軸のdeadzone（ゼロ範囲）サポート
    - 自律移動中のゲームパッド入力自動ブロック
- 協調搬送
    - `/collab/run_vel` および drive_vector の協調対応
    - 協調IDを `RobotStatus.msg` の `collab_id` として公開
    - コンテナ間通信切断を検知し協調グループを自動削除
    - 協調ナビゲーション停止時のLED/LCD/音声同期
- その他
    - BeaconのJetPack 6対応
    - オフラインローカルインストーラバンドル追加
    - Docker pullデーモン設定の導入
    - Safe Run用の協調/単体Config自動切替

### 機能改善

- ナビゲーションのstop/pause遅延時間を `snr_mux` と連動し、「発進します」音声再生中の即時停止を抑制
- SLSを瞬時ON/OFFしてERRORリセットを不要化
- MQTT client切断を警告レベルへ降格し、自動再接続リカバリを追加
- Pico瞬断を警告扱いとし、長時間断絶時はHSBリセットで復旧
- PICO接続タイムアウトをパラメータ化
- センサーデータQoSを `depth=1` のBestEffortに変更（輻輳低減）
- 回転時の障害物検出半径を独立設定化し、回転抑制を強化
- 運転ノード（Argus, VSLAM等）にCPU優先度を設定
- リモコン切断時に速度指示値を初期化
- point2dトピックごとのマスク適用およびデバッグ画像のポリゴン描画

### バグフィックス

- 自律協調時に意図せずエラー状態に入る不具合を修正
- 協調モード解除後、単体移動でモード状態が残留する不具合を修正
- `/collab/drive/stop` がプレフィックス付きで再配信されない不具合を修正
- 地図切替失敗後に走行再開ができない不具合を修正
- VSLAM再起動時に直前の地図が読み込まれない不具合を修正
- TagSLAMで新規地図作成・読込直後にタグが検出されない不具合を修正
- `/navigation/task/result` が同名タスク連続実行時に誤値を返す不具合を修正
- WebGUIからタスク編集ができない不具合を修正
- 低速な相対移動で停止・一時停止が効かない不具合を修正
- 2センサ式リフターの動作不具合、偏荷重・空荷でのリフトアップ失敗を修正
- 協調搬送中の角速度最大値計算誤りを修正
- Beaconで有線LAN＋無線LAN両接続時のホスト間通信失敗を修正
- SLS無効化時にsafe_runがOFFにならない不具合を修正

### 削除 / 互換性変更

- `triorb_dead_reckoning` のisam2ベース実装を廃止（代替としてIMUセンサ確認用バイパスログ機能を追加）
- `run_mapping` のmap analyzer連携機能を一時削除（改修中）
- 未使用だった `triorb_bridge` / `triorb_sls_wrapper` をビルド対象から除外
- `SlsBasicData` メッセージの `reserved` フィールドを削除（互換性注意）
- `RobotStatus.msg` に `collab_id` フィールド追加（互換性注意）
- 旧協調・FMSブリッジ起動スクリプト（`ENABLE_SYNC_REMOTE_GUI` 経由）を削除
