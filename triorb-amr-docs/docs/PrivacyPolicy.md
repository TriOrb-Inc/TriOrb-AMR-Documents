# TriOrb BASE 開発ガイド プライバシーポリシー（案）

制定日: 2026年3月27日

> 本文書は、`TriOrb-AMR-Documents` リポジトリおよび `TriOrb BASE 開発ガイド` の公開内容を前提に作成したドラフトです。実運用前に、株式会社TriOrbの実際の問い合わせ窓口、委託先、データ保持方針および法務レビューに合わせて調整してください。

[English version](./PrivacyPolicy.en.md)

## 1. 基本方針

株式会社TriOrb（以下「当社」といいます。）は、当社が提供または管理する `TriOrb BASE 開発ガイド`、`TriOrb-AMR-Documents` リポジトリおよびこれらに付随する公開ページ（以下、総称して「本サイト」といいます。）において取得する利用者情報を、個人情報の保護に関する法律その他関係法令に従い、適切に取り扱います。

## 2. 本ポリシーの対象

本ポリシーは、主に以下の場面に適用されます。

- 本サイトを閲覧した場合
- GitHub Issue、Discussion、Pull Request 等を通じて当社または本リポジトリ管理者に連絡した場合
- 当社が将来提供する ChatGPT App 等の非公開チャネルを通じて当社に連絡した場合
- 本サイトに掲載された外部リンク先へ遷移した場合に、当社がその導線を提供した範囲

外部サービス自体が独自に取得する情報については、各サービス提供者のポリシーが適用されます。

## 3. 取得する情報

当社または本サイトの運用基盤は、以下の情報を取得することがあります。

### 3.1 利用者が自ら提供する情報

- GitHub アカウント名、表示名、プロフィール情報
- Issue、Discussion、Pull Request、コメント、添付資料その他の投稿内容
- 当社が将来提供する ChatGPT App 等に入力された質問内容、要望、エラー報告、評価、添付資料その他の送信内容
- 問い合わせ時に利用者が任意に記載した連絡先、所属、環境情報その他の情報

### 3.2 本サイトの利用に伴い取得される情報

- IP アドレス、利用日時、閲覧した URL、参照元、ブラウザ種別、OS その他のアクセスログ相当情報
- GitHub、GitHub Pages その他のホスティング基盤が通常取得する技術情報
- 表示設定の保持等のためにブラウザのローカルストレージ等へ保存される設定情報

本サイトは MkDocs / Material for MkDocs を利用した静的ドキュメントサイトとして構成されており、画面表示設定の保持等のためにブラウザ側保存機能を利用する場合があります。

### 3.3 外部サービス経由で取得される可能性がある情報

本サイトには、GitHub、Notion、Box、当社コーポレートサイト、他の GitHub Pages および当社が将来利用する ChatGPT App の提供基盤等へのリンクまたは外部アセット参照が含まれます。利用者がこれらのサービスを利用した場合、当該サービス提供者がそれぞれのポリシーに従って情報を取得することがあります。

## 4. 利用目的

当社は、取得した情報を以下の目的で利用します。

- 本サイト、API 仕様、サンプル、変更履歴その他の公開情報を提供するため
- 問い合わせ、不具合報告、提案、投稿内容への対応を行うため
- ドキュメント品質の改善、構成見直し、保守および障害対応を行うため
- TriOrb 製品、関連ソフトウェア、ドキュメントおよびサポート品質の向上のために、問い合わせ傾向や改善要望を分析するため
- 不正利用、セキュリティ事故、権利侵害または運用上の問題を防止し、調査し、対応するため
- 法令遵守、権利保護または紛争対応のため

当社は、将来提供する ChatGPT App 等の非公開チャネルに寄せられた内容について、必要に応じて個人を直接特定できる情報を除去し、匿名化または統計化した上で、TriOrb 製品、関連ソフトウェア、ドキュメントおよびサポート品質の向上に利用することがあります。

一方、GitHub Issue や GitHub Discussions 等の公開投稿は、投稿者の GitHub アカウント名等が公開されるため匿名性は限定的です。当社は、これらの公開投稿についても、公開された範囲で、TriOrb 製品、関連ソフトウェア、ドキュメントおよびサポート品質の向上に利用することがあります。

## 5. 第三者提供および委託

当社は、法令に基づく場合を除き、利用者の個人データを本人の同意なく第三者へ提供しません。

ただし、本サイトのホスティング、ソースコード管理、Issue / Discussion 運営、ドキュメント配信、ChatGPT App 等の提供基盤または外部資料の提供にあたり、GitHub、GitHub Pages、Notion、Box その他のサービスを利用する場合があります。この場合、各サービス提供者がその役割に応じて情報を取り扱うことがあります。

## 6. 安全管理

当社は、取得した情報について、漏えい、滅失、毀損、不正アクセスその他のリスクに対し、合理的な範囲で必要かつ適切な安全管理措置を講じるよう努めます。

一方で、GitHub Issue や GitHub Discussions など公開スペースに利用者が自ら投稿した情報は、公開情報として第三者から閲覧される可能性があります。公開投稿では、個人情報、秘密情報、顧客情報または認証情報を記載しないでください。

## 7. 保存期間

当社は、取得した情報を、利用目的の達成に必要な期間または法令上必要な期間に限って保持します。ただし、GitHub 上の公開投稿や外部サービス上の記録は、当社の削除権限や保存期間の管理が及ばない場合があります。

## 8. 利用者の権利

利用者は、法令の定めに従い、当社が保有する自己の個人情報について、開示、訂正、追加、削除、利用停止等を求めることができます。

具体的な請求方法および本人確認方法は、`info@triorb.co.jp` または当社が別途指定する窓口によります。公開された GitHub Issue や GitHub Discussions は、個人情報に関する請求窓口としては適しません。

## 9. Cookie 等

本サイト自体は静的コンテンツを中心に構成されていますが、GitHub、GitHub Pages、当社コーポレートサイトまたはリンク先サービスにより Cookie 等が利用される場合があります。また、本サイトの表示設定保持等のためにブラウザのローカルストレージ等が利用されることがあります。

利用者は、ブラウザ設定により Cookie 等の保存を制御できますが、その結果、一部機能や表示設定の保持が利用できなくなる場合があります。

## 10. 外部リンク

本サイトには、Notion、Box、GitHub、当社コーポレートサイト、他の GitHub Pages サイト等へのリンクが含まれます。リンク先での個人情報の取扱いについては、当該外部サービスまたは運営主体のポリシーをご確認ください。

## 11. 未成年者

未成年の利用者は、必要に応じて法定代理人の同意を得た上で本サイトを利用してください。

## 12. 改定

当社は、法令改正、運用変更または本サイトの内容変更に応じて、本ポリシーを改定することがあります。重要な変更がある場合は、本サイト上での掲示その他適切な方法により周知します。

## 13. 問い合わせ窓口

本ポリシーに関する問い合わせ窓口は、`info@triorb.co.jp` とします。公開リポジトリ上での一般的な API 問い合わせと、個人情報に関する非公開の問い合わせ窓口は分けて運用します。
