利用規約¶

制定日: 2026年3月27日

This document is a draft prepared based on the current public content of the TriOrb-AMR-Documents repository and the TriOrb BASE Developer Guide. Before production use, it should be reviewed and adjusted to match TriOrb Inc.'s actual business operations, contract structure, sales model, and legal review.

適用範囲¶

本利用規約（以下「本規約」といいます。）は、株式会社TriOrb（以下「当社」といいます。）が提供または管理する以下の公開情報および関連ページの利用に適用されます。

  • The documentation site published as the TriOrb BASE Developer Guide

  • The TriOrb-AMR-Documents repository and related GitHub pages introduced by us

  • Cases where a future ChatGPT App or other support channel provided by us directs users to this site or GitHub pages

  • API specifications, samples, manuals, change logs, external links, and related materials published through the above

外部サービス、外部サイトまたは別リポジトリには、それぞれ独自の利用条件が適用される場合があります。

本サイトの目的¶

本サイトは、TriOrb BASE および関連する自律移動・協調移動・制御用ソフトウェアに関する開発者向け情報、API 仕様、サンプルコード、参照資料等を提供することを目的としています。

本サイト上の情報は、利用者による評価、開発、検証および運用準備を支援するための参考情報であり、個別案件に対する保証、設計受託、保守契約または安全性保証を当然に構成するものではありません。

規約への同意¶

利用者は、本サイトを利用した時点で、本規約に同意したものとみなされます。本規約に同意できない場合は、本サイトの利用を中止してください。

利用条件¶

利用者は、法令、本規約、当社が別途提示するマニュアル、安全上の注意および第三者との契約条件を遵守した上で、本サイトを利用するものとします。

利用者は、以下の事項を理解した上で本サイトを利用するものとします。

  • Use of robots, ECUs, ROS 2 nodes, Web APIs, and related software requires separate hardware, network setup, operational procedures, and safety management

  • Any application of published API specifications or sample code to real hardware requires prior verification, hazard assessment, fail-safe design, and compliance checks under applicable laws and regulations

  • Access to unpublished repositories, binaries, credentials, or customer-specific settings is not granted merely because this site is publicly available

禁止事項¶

利用者は、本サイトの利用にあたり、以下の行為をしてはなりません。

  • Any act that violates laws or public order and morals

  • Any act that infringes the rights, interests, reputation, or credibility of us or any third party

  • Any act that imposes excessive load on this site, GitHub repositories, hosting infrastructure, or related services

  • Unauthorized access, vulnerability probing, bypass of authentication, abusive scraping, malware transmission, or any other security intrusion

  • Unauthorized acquisition, use, or disclosure of information, repositories, credentials, or customer environment data that we treat as non-public

  • Circumvention of robot safety mechanisms, control restrictions, or operational safeguards in a manner that causes dangerous behavior

  • Alteration, redistribution, or presentation of site content in a misleading way, including suggesting that modified content is official information from us

  • Posting personal information of third parties, confidential information, unlawful content, or otherwise inappropriate content to GitHub Issues, Discussions, Pull Requests, or similar channels

知的財産権¶

本サイトおよび本リポジトリに含まれる文書、画像、構成、ソフトウェア、商標、ロゴその他の権利は、当社または正当な権利者に帰属します。

本リポジトリのソースファイル、ドキュメント原稿その他の成果物について、別途ライセンス表示がある場合は当該表示が優先します。現時点では、本リポジトリのルートに Apache License 2.0 が置かれており、当該ライセンスが適用される範囲ではその条件に従って利用できます。

ただし、以下については本リポジトリのオープンソースライセンスとは別の条件が適用される場合があります。

  • The TriOrb and TriOrb BASE names, our logos, and other trademark-related elements

  • Materials published on external sites such as PDFs, Notion pages, Box pages, and GitHub Pages

  • Software, images, documents, and linked content owned by third parties

投稿コンテンツ¶

利用者が GitHub Issue、Discussion、Pull Request その他の方法で本リポジトリまたは関連ページに投稿した内容は、公開状態で閲覧されることがあります。公開したくない個人情報、機密情報または第三者情報は投稿しないでください。

利用者が本リポジトリへの取り込みを意図して行う提案、修正または貢献は、別段の合意がない限り、本リポジトリに適用されるライセンス条件および GitHub の利用条件に従って取り扱われるものとします。

当社が将来提供する ChatGPT App 等の非公開チャネルに利用者が入力した問い合わせ、要望、エラー報告または改善提案について、当社は、個人を直接特定できる情報を可能な範囲で除去、匿名化または統計化した上で、TriOrb 製品、関連ソフトウェア、ドキュメントおよびサポート品質の向上に利用することがあります。

一方、GitHub Issue、Discussions、Pull Request 等への投稿は公開情報であり、投稿者の GitHub アカウント名その他の公開プロフィールが第三者に閲覧されます。これらの投稿の匿名性は限定的です。当社は、これらの公開投稿についても、公開された範囲で、TriOrb 製品、関連ソフトウェア、ドキュメントおよびサポート品質の向上に利用することがあります。

本サイトでは、GitHub、GitHub Pages、Notion、Box、当社コーポレートサイト、当社が将来提供する ChatGPT App の提供基盤その他の外部サービスへのリンク、画像参照またはホスティング機能を利用することがあります。これらの外部サービスは当社の管理外にあり、その利用には各サービス提供者の規約およびポリシーが適用されます。
