triorb_drive_vector¶

制御指令値からロボットの進行方向や停止・回転などの状態判定を行う

  • API

概要¶

駆動指令を進行方向・回転・停止といった離散的なベクトル表現へ変換し、表示制御や安全装置連携へ引き渡すためのパッケージです。

主な機能¶

  • 制御指令値からロボットの進行方向や停止・回転などの状態判定を行う。SLSやLED制御向け。

  • 閾値設定はdrive_vector.xml

更新履歴¶

1.2.4¶

  • 単体/協調のベクトル出力継続時間を見直し、不要なリセットや送信切れを修正

  • 協調時のバインド姿勢反映やレビュー指摘対応により、進行方向通知の安定性を改善

1.2.3¶

  • 協調移動向けのベクトル出力に対応

  • /drive/std_vector2 を追加し、下流ノード向けの進行方向表現を整理

1.1.0¶

  • SLSや表示制御向けの進行方向判定ノードとして初版を作成

  • 閾値設定と基本ドキュメントを追加
